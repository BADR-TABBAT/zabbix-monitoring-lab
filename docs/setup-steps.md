# Setup Steps

## 1. Prepare the lab environment
- Identify the server or VM used for monitoring
- Prepare the monitored hosts
- Verify network connectivity

## 2. Access Zabbix
- Connect to the Zabbix web interface
- Review the dashboard
- Explore the monitoring sections

## 3. Add hosts
- Add one or more hosts
- Assign groups if needed
- Configure interfaces

## 4. Configure monitoring items
- Define basic checks
- Review metrics collection
- Verify item status

## 5. Create triggers and alerts
- Define simple alert conditions
- Test the monitoring logic
- Observe alert behavior

## 6. Review dashboards
- Check host health
- Explore availability views
- Monitor system activity
