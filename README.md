# Zabbix Monitoring Lab

## Project Overview
This project presents a basic monitoring lab built with Zabbix to supervise systems, hosts, and services.

## Objectives
- Understand the role of monitoring in IT infrastructure
- Learn how to add hosts and configure monitoring items
- Explore dashboards, triggers, and alerts
- Improve visibility on system and service availability

## Environment
- Zabbix
- Linux / Virtual lab
- Networked hosts
- Basic monitoring setup

## Tasks Completed
- Installed or prepared the Zabbix environment
- Added monitored hosts
- Configured monitoring items
- Created triggers and alerts
- Explored dashboards and availability views

## Repository Structure
- `README.md` → Main project presentation
- `docs/overview.md` → General overview of the lab
- `docs/setup-steps.md` → Main configuration steps
- `docs/notes.md` → Challenges, issues, and lessons learned
- `docs/screenshots/` → Visual documentation

## Key Learning Outcomes
- Better understanding of monitoring concepts
- Familiarity with hosts, items, triggers, and alerts
- Improved documentation and lab organization skills

## Important Note
This repository contains a personal/lab version of a monitoring project. Any sensitive or real production information has been removed or anonymized.
